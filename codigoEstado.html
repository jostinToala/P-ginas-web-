Códigos de respuesta HTTP

Los códigos de respuesta HTTP son códigos de tres dígitos enviados por un servidor web en respuesta a una solicitud realizada por un cliente. Estos códigos proporcionan información sobre el estado de la solicitud y ayudan a diagnosticar problemas de comunicación. A continuación, se presentan algunos ejemplos de códigos de respuesta comunes:

  1. 1xx: Respuestas informativas
  2. 2xx: Respuestas exitosas
  3. 3xx: Redirecciones
  4. 4xx: Errores del cliente
  5. 5xx: Errores del servidor
