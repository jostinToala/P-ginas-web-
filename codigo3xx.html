Códigos 3xx - Redirecciones

Los códigos de respuesta 3xx indican que se requiere una acción adicional para completar la solicitud. Estos códigos se utilizan para redirigir al cliente a una ubicación diferente. Algunos ejemplos comunes son:

  • 301 Moved Permanently
  • 302 Found
  • 307 Temporary Redirect

301 Moved Permanently

El código de respuesta 301 Moved Permanently indica que la ubicación del recurso solicitado ha sido permanentemente cambiada. El cliente debe actualizar su URL para futuras solicitudes.

302 Found

El código de respuesta 302 Found indica que la ubicación del recurso solicitado ha sido temporalmente cambiada. El cliente debe continuar utilizando la URL original para futuras solicitudes.

307 Temporary Redirect

El código de respuesta 307 Temporary Redirect indica que la solicitud debe ser redirigida temporalmente a otra URL. A diferencia del código 302, el cliente debe continuar utilizando la misma URL para futuras solicitudes.
