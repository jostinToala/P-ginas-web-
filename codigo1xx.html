Códigos 2xx - Respuestas exitosas

Los códigos de respuesta 2xx indican que la solicitud se ha procesado con éxito y se ha obtenido el resultado esperado. Algunos ejemplos comunes son:

  1. 200 OK
  2. 201 Created
  3. 204 No Content

200 OK

El código de respuesta 200 OK indica que la solicitud ha sido exitosa. Es el código de respuesta más común y se utiliza cuando no hay problemas con la solicitud y el servidor devuelve correctamente el contenido solicitado.

201 Created

El código de respuesta 201 Created se utiliza para indicar que la solicitud ha sido exitosa y que se ha creado un nuevo recurso en el servidor. Por ejemplo, después de enviar un formulario de registro, se puede devolver este código para indicar que se ha creado una nueva cuenta de usuario.

204 No Content

El código de respuesta 204 No Content se utiliza para indicar que la solicitud ha sido exitosa, pero no se devuelve ningún contenido en la respuesta. Este código se usa a menudo en solicitudes de eliminación o actualización, donde no es necesario devolver información adicional.
